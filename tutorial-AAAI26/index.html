Why This Tutorial Matters

The era of foundation models has created an adaptation bottleneck: how can we efficiently repurpose massive pre-trained models for specific tasks without extensive fine-tuning? Neural network reprogrammability offers a solution through input space transformations and output alignment techniques.

This tutorial introduces a unified framework encompassing three core paradigms:

  • Model Reprogramming: Learning input transformations to repurpose frozen models
  • Prompt Tuning: Learning continuous or discrete prompts to guide model behavior
  • Prompt Instruction: Using natural language instructions and few-shot examples

You'll gain both theoretical understanding and practical skills to implement these techniques in your own research and applications.
